Bezwaar tegen uitwisseling van gegevens met PALGA
De resultaten van het bevolkingsonderzoek worden vanuit het laboratorium uitgewisseld met de landelijke PALGA databank (Pathologisch Anatomisch Landelijk Geautomatiseerd Archief). Wilt u hier bezwaar tegen maken, dan moet u dit bezwaar indienen bij het laboratorium. U vindt hiervoor een formulier op de website van de screeningsorganisatie (pagina ‘Bezwaar’).
