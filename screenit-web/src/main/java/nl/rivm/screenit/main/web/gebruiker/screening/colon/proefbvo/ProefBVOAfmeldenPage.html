Definitief afmelden voor proefbevolkingsonderzoek darmkanker

Met dit formulier kun je clienten afmelden als proefbevolkingsonderzoek voor darmkanker. Door het uploaden van een csv, met twee velden genaamd anummer, geboortedatum(yyyyMMdd) kun je de clienten selecteren. De afmeldbrief kan alleen in de vorm van een tiff of pdf worden geupload.
