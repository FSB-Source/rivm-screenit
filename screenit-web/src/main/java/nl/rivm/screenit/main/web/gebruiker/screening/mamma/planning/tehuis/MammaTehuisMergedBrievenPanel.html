Voor cliënt(en) is een uitnodiging aangemaakt. Deze cliënt(en) behoren/behoort op dit moment bij een project, waardoor er vervangende brieven zijn/brief is aangemaakt. Deze worden uiterlijk de ochtend van de volgende werkdag in de projectbrieven genereren batch samengevoegd en klaargezet voor afgdrukken (regulier proces).
