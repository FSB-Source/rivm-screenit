Wachtwoord wijzigen

Uw nieuwe wachtwoord moet voldoen aan de volgende eisen:
  • 'Wachtwoord' moet minstens 8 karakters hebben
  • 'Wachtwoord' moet minimaal 1 cijfer hebben
  • 'Wachtwoord' moet minimaal 1 hoofdletter hebben
  • 'Wachtwoord' moet minimaal 1 niet-alfanumeriek teken hebben
  • Uw gebruikersnaam mag niet voorkomen in uw wachtwoord
