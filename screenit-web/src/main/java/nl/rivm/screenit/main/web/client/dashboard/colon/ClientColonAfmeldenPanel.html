Afmelden bevolkingsonderzoek darmkanker

U ontvangt over twee jaar opnieuw een uitnodiging voor het bevolkingsonderzoek darmkanker, tenzij u dan 76 jaar of ouder bent. Wij waarderen het als u onder in dit scherm de reden van uw afmelding aangeeft. Het buisje kunt u in de afvalbak doen, ook alle overige materialen kunt u wegdoen.

U ontvangt een schriftelijke bevestiging van uw keuze.
Wilt u zich op een later moment opnieuw aanmelden voor het bevolkingsonderzoek, dan kan dit via Mijn Bevolkingsonderzoek of via de informatielijn van de regionale screeningsorganisatie.
