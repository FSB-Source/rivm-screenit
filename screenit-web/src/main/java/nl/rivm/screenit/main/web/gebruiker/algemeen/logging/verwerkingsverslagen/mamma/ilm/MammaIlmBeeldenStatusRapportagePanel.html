ILM BK beelden met status te lang op "te verwijderen"

De ILM BK batch heeft naar aanleiding van een bezwaar of verlopen bewaartermijn een verwijder-bericht gestuurd naar IDS7.
Voor onderstaande onderzoek(en) is, na opnieuw proberen, geen verwijder-bevestiging ontvangen vanuit IDS7.
Met de verwijderen forceren knop wordt de status voor de betreffende beelden alsnog aangepast van 'Te verwijderen' naar 'Verwijderd'. Dit pas na fysieke controle in IDS7 door IMS beheer.
