Uitstel aanvragen

U kunt uw deelname aan het bevolkingsonderzoek baarmoederhalskanker uitstellen.
Stelt u uw deelname uit omdat u zwanger bent? U ontvangt ongeveer zes maanden na de door u opgegeven vermoedelijke bevallingsdatum een nieuwe uitnodiging.
Stelt u om een andere reden uw deelname uit, dan kunt u zelf aangeven wanneer u een nieuwe uitnodiging wilt ontvangen.
